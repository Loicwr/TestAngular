Formulaires avec FormsModule

Pour créer des formulaires simples et dynamiques, lier les données à des champs, valider les saisies et gérer l'envoi.

1. Importer FormsModule dans appModule

Angular ne peut pas utiliser [(ngModel)] sans ce module. Il active la gestion des formulaires template-driven.

Importer le module : FormsModule dans mon fichier .ts dans les [] au niveau de imports

---------------------------------------------------------------------

2. Lier des champs avec [(ngModel)]

Cela permet de lier une variable TypeScript à un champ HTML, dans les deux sens (saisie utilisateur ↔ variable).

Bonjour {{ prenom }} !

Tu vois en temps réel ce que l'utilisateur tape dans l'interface !

---------------------------------------------------------------------

3. Gérer l'envoi du formulaire avec (ngSubmit)

C'est l'événement qui s'active quand l'utilisateur clique sur "Envoyer". Il permet d'exécuter une méthode.

Le name="..." est obligatoire pour que Angular suive le champ.

---------------------------------------------------------------------

4. Ajouter des validations de base

Pour empêcher l'envoi si les données ne sont pas valides (ex : vide, trop court…).

Angular reconnaît les validations HTML (required, minlength, pattern, etc.) et agit automatiquement.

---------------------------------------------------------------------

5. Afficher les erreurs de validation avec *ngIf

Pour informer l'utilisateur quand il fait une erreur (champ vide, trop court…).

Le prénom est requis. Minimun 3 caratères.

.invalid = champ incorrect, .touched = l'utilisateur a cliqué dans le champ au moins une fois.
