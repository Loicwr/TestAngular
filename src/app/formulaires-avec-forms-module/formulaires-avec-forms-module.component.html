Formulaires avec FormsModule

Pour créer des formulaires simples et dynamiques, lier les données à des champs, valider les saisies et gérer l'envoi.

1. Importer FormsModule dans appModule

Angular ne peut pas utiliser [(ngModel)] sans ce module. Il active la gestion des formulaires template-driven.

Importer le module : FormsModule dans mon fichier .ts dans les [] au niveau de imports

---------------------------------------------------------------------

2. Lier des champs avec [(ngModel)]

Cela permet de lier une variable TypeScript à un champ HTML, dans les deux sens (saisie utilisateur ↔ variable).

Bonjour {{ prenom }} !

Tu vois en temps réel ce que l'utilisateur tape dans l'interface !
