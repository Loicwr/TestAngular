Directive Structurelles

1. *ngIf

Afficher un bloc HTML seulement si une condition est vraie.

Bienvenue, vous êtes connecté !

Si " isLoggedIn " = false, alors balise p ne s'affiche pas
---------------------------------------------------------------------

2. *ngIf ... else - Affichage conditionnel avec fallback

Afficher un contenu alternatif si la condition est fausse.

Bienvenu Admin

Accès refusé.
Si " isAdmin " = false, il affiche le contenu de ng-template
---------------------------------------------------------------------

3. *ngFor - Répéter un bloc pour chaque élément d'une liste

Créer dynamiquement des éléments HTML à partir d'un tableau.

  • {{ animal }}
Affiche une liste avec chaque animal.

Syntaxe détaillé

*ngFor =" let item of items; let i = index "

let item : Crée une variable pour chaque élément

of item : Tableau à parcourir

let i = index : Donne l'index(0, 1, 2, ...) de l'élément

{{ i + 1 }}. {{ fruit }}

Conseil pratiques

  • Toujours déclarer tes variables dans le fichier .ts du composant.
  • Les ng-template ne s'affichent jamais seuls : ils servent uniquement avec des else.
  • Pour le style dynamique, combine *ngIf avec [ngClass], [style] ou [hidden] si besoin.
