Créer et utiliser un service Angular

1. Générer un service avec ng generate service

La commande Angular CLI permet de créer automatiquement la structure d'un service avec tous les fichiers nécessaires.
Lorsque tu exécutes ng generate service services/mon-service,
Angular génère un fichier TypeScript contenant la classe du service avec le décorateur @Injectable() déjà configuré,
ainsi qu'un fichier de test unitaire correspondant. Cette commande crée également le dossier services s'il n'existe pas encore,
ce qui permet d'organiser proprement l'architecture de ton application en regroupant tous les services dans un même répertoire.
