Qu'est-ce qu'un pipe en Angular ?

Un pipe (qu'on peut traduire par "tuyau") sert à transformer des données dans le HTML, sans changer les vraies valeurs dans ton fichier TypeScript (.ts). Imagine que tu as une valeur brute (comme une date ou un texte en minuscule), et que tu veux simplement l'afficher différemment à l'écran (majuscule, format de date lisible, etc.).

Plutôt que d'écrire une fonction à chaque fois, Angular fournit des pipes prêts à l'emploi.

Syntaxe de base : dans une double Accolade ( alt + parenthèse ouvrante ) : valeur | nomDuPipe

---------------------------------------------------------------------

1. Pipe intégrés à Angular

uppercase et lowercase

angular devient avec uppercase {{ nom | uppercase }}

ANGULAR devient avec lowercase {{ nom | lowercase }}

---------------------------------------------------------------------

Quel est la date (date) : {{ aujourdhui | date }}

Quel est la date (shortDate) : {{ aujourdhui | date: 'shortDate' }}

Quel est la date (fullDate): {{ aujourdhui | date: 'fullDate' }}
